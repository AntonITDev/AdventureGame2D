# AdventureGame2D
2D web game


CONTROLS:

Move
* A - left;
* D - right;
* W - up;
* S - bottom;

* ESC - Open menu;
* E - activate



Features:
* Settings;
* Cheasts and keys;
* Random level generation;
* Adaptive for mobile playing;
* Save results;
* Custom change characters;
* Add Battles
* Inventory
* Items (Weapon and Armor)
* Random Market
